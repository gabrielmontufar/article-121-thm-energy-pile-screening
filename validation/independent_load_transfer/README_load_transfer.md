# Independent Load-Transfer Comparator

This folder contains a deliberately simple elastic load-transfer-style comparator. It is independent from the reduced THM benchmark and is used to check whether a lower-complexity method preserves the same serviceability classification.

Run:

```powershell
python .\validation\independent_load_transfer\run_load_transfer_comparator.py
```

The comparator is not a field-calibrated model. It supports screening-level agreement checks only.
